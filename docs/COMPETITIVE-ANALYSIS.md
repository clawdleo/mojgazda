### Legal Framework for Rating Individuals

#### European/Balkan Context
1. **GDPR (General Data Protection Regulation)**
   - Managers' names = personal data
   - Processing requires legal basis
   - **Legitimate interest** defense: public interest in workplace transparency
   - **Right to be forgotten** requests could be used to remove profiles
   - RateMyProfessors survives in EU under "freedom of expression" exception

2. **Defamation Laws (Croatia, Slovenia, Serbia, BiH)**
   - Truth is always a defense
   - Opinions are generally protected (not facts)
   - "My manager is terrible" = opinion (protected)
   - "My manager steals money" = factual claim (must be provable)
   - Anonymous sources make enforcement difficult

3. **Precedent: kununu.com (Germany/Austria)**
   - German employer review site with 5M+ reviews
   - Courts have consistently upheld anonymous employer reviews
   - Key: platform must respond to complaints, have review process

4. **Croatian Labor Law**
   - Workers have right to report workplace conditions
   - Whistleblower protection (EU Whistleblower Directive 2019/1937)
   - Public interest in workplace transparency recognized
